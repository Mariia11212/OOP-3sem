# Лабораторна робота №X: Об'єктно-орієнтовані структури даних та геометрія на площині

## Виконаний функціонал

| Перелік | Що реалізовано | Додатково |
|---------|----------------|------------|
| **Перелік 1: Структури даних та алгоритми** | - Списки (`MyLinkedList<T>`) та циклічні списки (`MyCircularList<T>`)<br>- Масивні списки (`MyDynamicArrayBasedList<T>`) <br>- Пошук елементів за значенням (`Contains`, `IndexOf`) та за індексом (`GetAt`) <br>- Пошук першого елемента за умовою (`FindFirst`) <br>- Операції додавання, вставки, видалення, очищення списку <br>- Підтримка різних типів: примітивні, бібліотечні та власні класи | - Генерація текстового представлення списку <br>- Підтримка практичної демонстрації роботи зі списками в `Program.cs` |
| **Перелік 2: Геометрія на площині** | - Базові фігури: точки (`Point`), відрізки (`Segment`), кола (`Circle`) <br>- Складені фігури: багатокутники (`Polygon`), прямокутники (`Rectangle`) <br>- Перетворення: переміщення (`Translation`), обертання (`Rotation`), масштабування (`Scaling`) <br>- Алгоритми обчислення площі, периметра, центру мас/центроїду <br>- Додаткові функції: відстань між точками, перевірка точки в колі, середина відрізка | - Підтримка комбінацій перетворень <br>- Інтеграція з власними структурами даних для колекцій фігур |
| **Додатково** | - 18 класів/типів, понад 20 полів, понад 50 нетривіальних методів <br>- 3 ієрархії успадкування <br>- 4 незалежні випадки поліморфізму (динамічний та статичний через Generics) <br>- Правильна інкапсуляція та використання абстрактних класів/інтерфейсів | - Демонстрація роботи компонентів у `Program.cs` <br>- Підтримка генерації випадкових даних та текстового представлення для списків та фігур |
