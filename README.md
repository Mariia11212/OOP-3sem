# Лабораторна робота №X: Об'єктно-орієнтовані структури даних та геометрія на площині

## Виконаний функціонал

### Перелік 1: Структури даних та алгоритми
- **Списки та циклічні списки**:
  - Реалізація на основі зв'язних списків (`MyLinkedList<T>`).
  - Реалізація на основі масивів змінної довжини (`MyDynamicArrayBasedList<T>`).
  - Циклічний список (`MyCircularList<T>`).
- **Пошук елементів**:
  - За значенням (`Contains`, `IndexOf`).
  - За індексом (`GetAt`).
  - Першого елемента за заданою умовою (`FindFirst(Func<T,bool>)`).
- **Додаткові можливості**:
  - Додавання, вставка, видалення елементів.
  - Очищення списку (`Clear`).
  - Генерація текстового представлення списку для виводу.
  - Підтримка роботи з різними типами даних: примітивні (`int`, `double`), бібліотечні (`string`), а також власні класи (геометричні фігури).

---

### Перелік 2: Геометрія на площині
- **Базові фігури**:
  - Точки (`Point`), відрізки (`Segment`), кола (`Circle`).
- **Складені фігури**:
  - Багатокутники (`Polygon`), прямокутники (`Rectangle`).
- **Перетворення фігур**:
  - Переміщення (`Translation`), обертання (`Rotation`), масштабування (`Scaling`).
  - Підтримка застосування перетворень до будь-якої фігури.
- **Алгоритми обчислення**:
  - Площа та периметр фігур.
  - Центр мас, центроїд.
  - Додаткові функції: обчислення відстані між точками, перевірка входження точки в коло, знаходження середини відрізка.

---

### Додатково виконано
- **Вимоги ООП та архітектура**:
  - 18 класів/типів, більше 20 полів, понад 50 нетривіальних методів.
  - 3 ієрархії успадкування:
    1. Геометричні фігури (`IShape -> Shape -> конкретні фігури`).
    2. Перетворення (`ITransformation -> конкретні перетворення`).
    3. Структури даних (`IDataList -> конкретні реалізації списків`).
  - 4 незалежні випадки поліморфізму: динамічний для фігур, перетворень, списків; статичний через Generics (`TransformationManager`).
  - Правильна інкапсуляція (приватні поля, властивості з `private set`).
  - Активне використання абстрактних класів та інтерфейсів (`Shape`, `IShape`, `ITransformation`, `IDataList`).
- **Інтеграція структур даних та геометрії**:
  - Колекції фігур (`IShape`, `Rectangle`, `Circle`) зберігаються та обробляються через власні списки (`MyDynamicArrayBasedList<T>`), що демонструє реальне застосування розроблених компонентів.
- **Розширений функціонал геометрії та списків**:
  - Багатокутники як узагальнена фігура, з якої успадковується прямокутник.
  - Підтримка трьох типів перетворень та їх комбінацій.
  - Кожен список підтримує повний набір операцій та пошук елементів за різними критеріями.
  - Можливість генерації випадкових даних та виводу у текстовому вигляді.
